# Projeto

Hoje finalizei o projeto da tela de login, apesar de ser um projeto simples winforms, foi complicado pois tive que pesquisar e aprender muitas coisas, a parte mais complexa dele foi fazer os comandos Sql funcionar perfeitamnte, tive que pesquisar bastante em vídeos e documentos tanto para a integtração com o banco de dados quanto para fazer os comandos Sql funcionar da maneira que deveriam. 

Apesar de tudo, os unicos problemas que tive foi em questão de tempo, pois realmente precisa pesquisar muitas coisas, o que acabou por um bom tempo sendo bem desgastante para mim, tive que faze diversas pausas, mas no final deu tudo certo.

Considerie já o projeto finalizado, tendo os campos, as mascaras em campos necessários, notificares de erros, verificações de campos faltando, senhas erradas, limitador de erros ao logar e um log de login.

Com o passar dos dias os assuntos mais complicados com o Sql, vai ficar salvo em um arquivo na parte de fundamentos futuramente, por ser algo bem mais teorico não fico totalmente focado em documentar exatamente quando utilizo, vejo dessa forma que utilizo até mesmo como uma forma de revisão dos conceitos já aprendidos, para depois revisados futuramente.
