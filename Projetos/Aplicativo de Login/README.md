# Aplicativo de Login

Essa apliacação tem como objetivo estudar as estruturas basicas de criação de um app desktop utilizando o winforms, além de explorar as possiveis verificações que podem ser feitas tudo dentro de um login simples, ideias de banco de dados e outras aplicações.

Essa vai ser uma aplicação simples inicial onde planejo terminar ela ainda nessa primeira semana de estudos, continuando ou aprimorando-a futuramente nas proximas semanas.
