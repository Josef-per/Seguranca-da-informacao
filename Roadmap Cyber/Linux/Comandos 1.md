# Comandos Linux

Anotações do estudo inicial de comandos Linux.

- **ls**  
Lista os arquivos e pastas do diretório atual.

- **cd**  
Troca de diretório.  
Uso: `cd caminho/do/diretorio`  
Exemplo: `cd Desktop`  
Se o nome tiver espaços, usar aspas: `cd "Diário Técnico"`

- **pwd**  
Mostra o caminho completo do diretório atual, útil para saber onde estou no sistema.

- **mkdir**  
Cria diretórios.  
Se o nome tiver espaços, deve ser colocado entre aspas.  
Também é possível criar várias pastas de uma vez.

- **cat**  
Exibe o conteúdo de arquivos de texto diretamente no terminal.

- **nano**  
Editor de texto no terminal.  
Abre arquivos existentes ou cria novos arquivos caso não existam.

- **man**  
Exibe o manual de ajuda de um comando, funcionando como uma documentação oficial.

- **chmod**  
Altera permissões de arquivos, especialmente para permitir a execução de scripts (`.sh`).

- **Execução de programas**  
Para executar um arquivo no diretório atual, normalmente é necessário usar `./nome_do_programa`, ou garantir que ele esteja no `$PATH`.
